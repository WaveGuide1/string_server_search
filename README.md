## Project Setup

### Requirements
```
- Python 3.10+
- Pip
- Virtualenv
```
### App Setup (Ubuntu System)
- **sudo apt update**
- **sudo apt install python3 python-pip python3-venv**
- **python3 -m venv name**
- **source name/bin/activate**
- **pip install -r requirements.txt**

### Running the Server
- **python3 src/app_server.py**
### Running the Client
- **python3 src/client.py**
